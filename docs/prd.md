# 모바일 청첩장 기획서

## 프로젝트 개요

친구의 모바일 청첩장을 제작한다.

전체적인 방향은 **깔끔하고 오래 보아도 질리지 않는 디자인**을 목표로 하며, 과한 애니메이션이나 화려한 효과는 사용하지 않는다. 사진이 가장 돋보일 수 있도록 여백을 충분히 활용하고, 섹션마다 명확한 구분을 두어 가독성을 높인다.

---

# 디자인 컨셉

## 키워드

- Clean
- Minimal
- Elegant
- Warm
- Spacious

## 메인 컬러

- White (#FFFFFF)
- Light Gray (#F7F7F7)
- Beige (#F6F2ED)
- Text : #333333
- Point : #B48A64

전체적으로 채도가 낮고 따뜻한 느낌을 유지한다.

---

# 디자인 원칙

- 여백을 넉넉하게 사용한다.
- 폰트 크기와 두께를 통해 정보의 우선순위를 표현한다.
- 카드 형태의 UI를 최소화한다.
- 그림자 사용은 최소화한다.
- 섹션마다 충분한 상하 여백을 둔다.
- 섹션이 바뀔 때 배경색을 교차하여 구분감을 준다.

예시

```
White

↓

Light Gray

↓

White

↓

Beige

↓

White
```

---

# 애니메이션

전체적으로 매우 절제된 애니메이션만 사용한다.

사용

- fade-up
- fade-in
- opacity transition

미사용

- 인트로 애니메이션
- 로딩 애니메이션
- 타이핑 효과
- 패럴랙스
- 확대 효과
- 회전 효과
- 카운트다운
- 기타 화려한 인터랙션

스크롤 시 자연스럽게 나타나는 정도만 적용한다.

# 섹션 구성

## 1. Main Hero

### 이미지

```
img/main.png
```

전체 화면 기준 약 80~90vh

이미지 위에 오버레이를 적용하여 텍스트 가독성을 높인다.

### 표시 내용

```
강동욱
♡

여다은

2026.09.06 SUN

PM 03:00

월드컵 컨벤션
임페리얼 볼룸 홀
```

버튼 없음.

스크롤 유도 아이콘만 하단에 배치한다.
